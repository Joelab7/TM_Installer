## Quick Installation (Windows)

1. **Download** the latest version of the application
2. **Run** `install.bat` or the file `install` as an administrator
3. A launcher (Python(TM) Launcher) will be created on your desktop
4. Double-click on the launcher to launch the application

### Prerequisites!

- Telegram account with connected phone number
- Retrieve the API keys linked to the Telegram number (available on [my.telegram.org](https://my.telegram.org/))
- Read the help parameter of the application for more information on configuring the application with the API keys.
